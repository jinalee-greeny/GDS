# Output Grouping + Style-Write Fix Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Reorganize the plugin's flat output checkboxes into a 6-category tree (master + nested toggles, styles folded in), drop shadow from Variables (Effect-Style-only), and fix the styles-not-created bug by moving style lookups to the async Figma API + surfacing failure reasons.

**Architecture:** `core/figma-map.js` removes `shadow` from the variable groups. `plugin/code.src.js` switches the two style lookups to `getLocalTextStylesAsync()`/`getLocalEffectStylesAsync()` (makes `applyEffectStyles` async). `plugin/ui.template.html` replaces the three flat target sections with a category tree whose leaves toggle `selection` (variable groups) / `targets.textStyles` / `targets.effectStyles`, and the CTA surfaces failure reasons. All regenerated by `python3 build_apps.py`.

**Tech Stack:** Vanilla JS (Figma plugin, no bundler), Python assembler, Node built-in test runner, local headless Chrome for UI smoke, Figma Plugin API.

## Global Constraints

- **Zero runtime/test deps**; plugin is plain JS; SSOT via `build_apps.py` (core/* canonical; `plugin/ui.html`, `plugin/code.js` generated, drift-guarded; never hand-edit generated files).
- **Do NOT modify** `core/token-core.js`, `core/studio-ui.js`, or the exporters; the web app + token-core parity must stay green.
- **shadow** is removed from `GROUP_KEYS` and `STRING_GROUPS`; `variablesPlan` no longer emits `shadow/*`. `effectStylePlan`/`textStylePlan`/`shadowToEffects` unchanged (they read `config.shadow` directly).
- **Style lookups async**: `figma.getLocalTextStylesAsync()` / `getLocalEffectStylesAsync()` (the deprecated sync forms are the suspected cause of styles never being created). `applyEffectStyles` becomes async; `applyPlan` awaits it. Variable lookups stay sync (they work).
- **Idempotent writes** preserved: variables + styles find-or-create by name; counter-after-write; per-item try/catch; `loadFontAsync` before each text style with graceful per-item failure.
- **Message contract** unchanged in shape: UI→main `{type:'apply', config, selection, targets:{variables,textStyles,effectStyles}, textOptions:{weights, family}}` → main `{type:'result', created, updated, failed:[{name,error}]}`. `selection` now excludes `shadow`; `targets.variables` = (selection non-empty).
- **Checkbox checked** set via the DOM `.checked` PROPERTY on fresh nodes — never passed through `el()` attrs (avoids the `setAttribute('checked', undefined)` always-checked bug).
- **Font input** keeps a stable id for focus/caret preservation across renders.
- **CTA** stays a fixed always-visible bottom bar; it must show failure reasons, not just a count.

**Category taxonomy (fixed):**
- Color → `color`
- Typography → `fontSize, fontFamily, fontWeight, lineHeight, letterSpacing` + Text Styles
- Spacing & Sizing → `space, radius, borderWidth`
- Effects → `opacity` + Effect Styles (shadow)
- Motion → `duration, easing`
- Layout → `zIndex, breakpoint`

- [ ] **Step 3: Switch style getters to async in `plugin/code.src.js`**

`applyEffectStyles` → async, await the getter:
```js
async function applyEffectStyles(config, acc) {
  var plan = FigmaMap.effectStylePlan(config);
  var existing = indexByName(await figma.getLocalEffectStylesAsync());
  for (var i = 0; i < plan.length; i++) {
    var item = plan[i];
    try {
      var s = existing[item.name], isNew = !s;
      if (isNew) { s = figma.createEffectStyle(); s.name = item.name; }
      s.effects = item.effects;
      if (isNew) acc.created++; else acc.updated++;
    } catch (e) { acc.failed.push({ name: item.name, error: (e && e.message) ? e.message : String(e) }); }
  }
}
```

```js
  var existing = indexByName(await figma.getLocalTextStylesAsync());
```
(rest of `applyTextStyles` unchanged; it's already async.)
`applyPlan` — await the effect writer:
```js
async function applyPlan(config, selection, targets, textOptions) {
  var acc = { created: 0, updated: 0, failed: [] };
  targets = targets || { variables: true };
  if (targets.variables) applyVariables(config, selection, acc);
  if (targets.effectStyles) await applyEffectStyles(config, acc);
  if (targets.textStyles) await applyTextStyles(config, textOptions || { weights: [], family: '' }, acc);
  return acc;
}
```

Leaf state helpers:
```js
function varOn(g) { return selection.indexOf(g) >= 0; }
function setVar(g, on) {
  var i = selection.indexOf(g);
  if (on && i < 0) selection.push(g);
  else if (!on && i >= 0) selection.splice(i, 1);
}
function styleOn(kind) { return kind === 'text' ? targets.textStyles : targets.effectStyles; }
function setStyle(kind, on) { if (kind === 'text') targets.textStyles = on; else targets.effectStyles = on; }
function catLeafStates(cat) {
  var arr = cat.vars.map(varOn).concat(cat.styles.map(styleOn));
  return { all: arr.every(Boolean), none: arr.every(function (x) { return !x; }) };
}
function setCategory(cat, on) {
  cat.vars.forEach(function (g) { setVar(g, on); });
  cat.styles.forEach(function (k) { setStyle(k, on); });
}
```
